Exercício 3

Edite a página HTML [link-propagation.html] que possui um div e dois links internos. O div, chamado de Div Pai, apresenta dois filhos, um link com comportamento normal e outro rebelde, chamado de link desobediente. O link normal, ao ser clicado, deve propagar o evento de clique para seu pai e então abrir o site referenciado no atributo href. Já o link desobediente não deve realizar nenhuma destas ações. Para isso, altere o código JavaScript para que o link desobediente apresente este comportamento rebelde.
Dica

Fazer uso das funções de parar a propagação de eventos e cancelar a ação padrão de um evento
